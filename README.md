# JcApiHelper
JcApiHelper

支持.Net Core 3.0

.Net Core 3.0请使用1.0.13版本

.Net Core 2.0请使用1.0.12版本

使用方法

1.在项目中,打开Nuget包管理,搜索Jc.ApiHelper,然后选择Jc.ApiHelper最新版本

2.在startup中,使用 app.UseJcApiHelper(); 引入ApiHelper

3.运行项目,访问http://localhost:5000/ApiHelper
